# TP 3 : Boucles & Tableaux en Java

Ce projet contient une série d'exercices en Java portant sur les concepts fondamentaux des boucles (for, while) et des tableaux (unidimensionnels et bidimensionnels).

## Liste des Exercices

1. **Exercice 1 : Série Harmonique**
   - Calcule la somme des n premiers termes de la série : 1 + 1/2 + 1/3 + ... + 1/n.
2. **Exercice 2 : Triangle Isocèle**
   - Dessine un triangle isocèle avec des étoiles, la hauteur étant fournie par l'utilisateur.
3. **Exercice 3 : Carré des Nombres Impairs**
   - Remplit un tableau avec les carrés des n premiers nombres impairs.
4. **Exercice 4 : Utilitaire de Tableaux 2D**
   - Fonctions pour initialiser, afficher et calculer la somme des éléments d'un tableau 2D.
5. **Exercice 6 : Fenêtre Glissante**
   - Calcule le nombre d'éléments distincts dans chaque fenêtre de taille k d'un tableau.
